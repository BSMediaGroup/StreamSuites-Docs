# 🌌 StreamSuites Platform Release — v0.5.0-alpha

`v0.5.0-alpha` continues the late-alpha hardening of the platform with major new foundations in economy, live status (Kick), public identity management, scoped progression, and broad surface parity work.

For the full detailed platform changelog, see the canonical master document:

**→ [v0.5.0-PROJECT-CHANGELOG.md](https://github.com/BSMediaGroup/StreamSuites/blob/main/v0.5.0-PROJECT-CHANGELOG.md)** (in the main runtime repository)

## Highlights

- Runtime/Auth: Market & Exchange, Kick live + webhooks, public identity system, scoped progression
- Creator: Complete Wheels system, improved triggers, identity tools
- Dashboard: Economy command center + public identity reconciliation workspace
- Public / FindMeHere: Market experience + directory surface maturation
- Developer Console: First-class shell parity and structured reporting
- Cross-surface: Strong movement toward runtime as source of truth + consistent auth gating
